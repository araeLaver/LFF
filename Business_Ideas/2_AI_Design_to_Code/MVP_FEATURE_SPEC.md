# MVP Feature Specification: AI 디자인-to-코드 변환

## 1. MVP 목표
> "AI가 Figma 디자인을 실제 개발자가 사용할 수 있는 수준의 React 컴포넌트 코드로 변환할 수 있는가?" 라는 기술적, 사업적 가설을 검증한다.
> 타겟 페르소나(React 개발자) 100명을 대상으로 테스트하여, "AI가 생성한 코드를 기반으로 작업하니, 기존 방식보다 개발 속도가 2배 이상 빨라졌다"는 피드백을 50% 이상 확보하는 것을 목표로 한다.

## 2. 주요 기능 (Features)

### 2.1. Figma 연동 및 디자인 임포트
*   **User Story 1:** 사용자는 Figma 계정으로 로그인하고, 자신의 Figma 파일 목록을 불러올 수 있다.
*   **User Story 2:** 사용자는 변환하고 싶은 Figma 파일의 URL과 특정 페이지(또는 프레임)를 선택하여 가져올 수 있다.
*   **Note:** Figma API의 OAuth 인증을 통해 사용자의 파일에 안전하게 접근한다.

### 2.2. AI 코드 생성 설정
*   **User Story 3:** 사용자는 코드를 생성하기 전, 변환할 프레임워크(`React` with `Next.js`)와 스타일링 방식(`Tailwind CSS`)을 선택할 수 있다.
*   **Note:** MVP에서는 가장 수요가 높은 `React + Tailwind CSS` 조합에 집중한다.

### 2.3. 코드 생성 및 실시간 미리보기
*   **User Story 4:** '코드 생성' 버튼을 누르면, AI가 선택된 Figma 프레임을 분석하여 컴포넌트 단위로 분리된 React 코드를 생성한다.
*   **User Story 5:** 사용자는 [코드 에디터], [웹 미리보기], [모바일 미리보기] 3단 뷰를 통해 생성된 코드가 실제 웹 환경에서 어떻게 렌더링되는지 즉시 확인할 수 있다.
*   **Note:** 코드 생성 과정(디자인 분석 중 -> 컴포넌트 구조화 중 -> 코드 생성 중)을 시각적으로 보여주어 사용자가 기다리는 동안 지루하지 않게 한다.

### 2.4. 코드 수정 및 다운로드
*   **User Story 6:** 사용자는 생성된 코드를 플랫폼 내의 웹 에디터에서 직접 수정하고, 수정 사항이 미리보기에 실시간으로 반영되는 것을 확인할 수 있다.
*   **User Story 7:** 사용자는 완성된 코드를 `.zip` 파일 형태로 다운로드하여 자신의 로컬 개발 환경에서 즉시 사용할 수 있다.
